00 · Brand Voice Profile
How Tom Talks — And How These Emails Sound
Every email is written in Tom's voice. These rules govern it.

Tom's name appears in every single client review. Not the company name — Tom. That is the competitive moat. These emails don't sound like "Greenbelt Event Rentals Marketing." They sound like Tom personally reaching out because he remembers you.

Tone
  • Warm, direct, personal — never salesy
  • Confident without being pushy
  • Conversational — writes like he talks
  • Proud of his work without bragging
  • Austin local energy — relaxed, genuine
Language Choices
  • Says "your event" not "your purchase"
  • Says "reach out" not "contact us"
  • Uses first names when known
  • Signs as "Tom" — not "Greenbelt Event Rentals"
  • Short sentences. No corporate filler.
What Tom Is Known For (Per Reviews)
  • "Responsive, friendly, thoughtful, and helpful"
  • "Went above and beyond"
  • "Professional and courteous"
  • "Made sure every detail was taken care of"
  • "Fantastic price" — transparency matters to him
What to Avoid
  • Exclamation marks every sentence
  • Marketing buzzwords ("amazing," "incredible," "exclusive")
  • Pressure or urgency tactics
  • Long paragraphs — people read on phones
  • Anything that sounds automated or templated
Important

Tom offers discounts for veterans, healthcare professionals, and teachers. Reference this where appropriate — it's a genuine differentiator and a values signal that resonates with his audience.

02 · Mailchimp Setup Instructions
Step-by-Step Configuration
Complete these in order before building any sequence.
Part A — Account & Plan
1
Confirm Standard Plan

Log in → click account name (top right) → Account & Billing → Billing. Confirm the plan shows Standard. If Free or Essentials, upgrade now.

Free plan lost all automation access in June 2025. Essentials caps automations at 4 steps. Standard is required. For 1,000 contacts, Standard costs approximately $20/month.

2
Set Sender Identity

Go to Account → Defaults. Set From Name to "Tom at Greenbelt Event Rentals" — not just the company name. Set Reply-To to Tom's direct email. Verify the sending domain under Account & Billing → Verified Domains.

Part B — Audience & Custom Fields
3
Add Custom Date Fields

Go to Audience → Manage Audience → Settings → Audience fields and *|MERGE|* tags. Scroll to bottom → Add A Field. Add these three fields — type must be Date for each:

  • Event Date → Tag: EVENTDATE
  • Last Booking Date → Tag: LASTBOOK
  • Inquiry Date → Tag: INQUIRYDT

Save after adding each. These are the trigger fields for all four automations.

4
Import Contacts by Segment

Go to Audience → Add Contacts → Import Contacts. Import each CSV file separately. On the import screen, map each column to the correct Mailchimp field — especially confirm the date columns map to the Date fields from Step 3.

After each import, apply the corresponding tag: WEDDING, CORPORATE, PERSONAL, or INQUIRY. All four imports go into the same Mailchimp audience.

5
Create Four Audience Segments

Go to Audience → Segments → Create Segment. Create one segment per tag:

  • Wedding Clients — Tag is WEDDING
  • Corporate Clients — Tag is CORPORATE
  • Personal Clients — Tag is PERSONAL
  • Inquiry Reactivation — Tag is INQUIRY
Part C — Building Automations
6
Build the Master Email Template First

Go to Content → Email Templates → Create Template. Build one clean mobile-responsive template: Greenbelt logo at top, body text area, single CTA button linking to greenbelteventrentals.com/contact, Tom's signature at bottom. Use this template for all 12 emails — only the copy changes.

7
Create Each Classic Automation

Go to Automations → Classic Automations → Create. Select Date Based. Name it clearly. Set the trigger field, timing, and segment filter per the sequence specs below. Add each email with the correct delay. Start sending only after all emails in the sequence are approved by Tom.

8
Wedding & Personal — Use Recurring Date Trigger

When configuring these two automations, select Recurring Date (not Specific Date) as the trigger type. This re-enters each contact into the sequence every year automatically. Corporate and Reactivation use different logic — see their individual setup notes below.

07 · What to Track After Launch
The Numbers That Actually Matter
Check monthly in Mailchimp Reports. Adjust copy on anything underperforming.
Open Rate
Target: ≥ 25%
Reports → Campaign → Overview. If below target, test a more personal subject line. First-name personalization in the subject can add 5–10%.
Click Rate
Target: ≥ 3%
If low, simplify the CTA. One button, one ask per email. No competing links.
Unsubscribe Rate
Target: ≤ 0.5%
If above 0.5%, reduce send frequency or examine tone. High unsubscribes means too much pressure or wrong timing.
Replies Received
Target: Any response is a win
Reply-to goes directly to Tom. Respond same day. Treat each reply as a warm lead who read the email and engaged.
New Bookings from Email
Target: ≥ 5 / month by Month 3
Ask new clients at booking: "How did you hear about us?" Track this manually if needed. This is the primary revenue metric.
